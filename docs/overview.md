# Discord Search

Discordの会話を自然言語で検索できるBot

## サービス概要

| 項目 | 内容 |
|------|------|
| 表示名 | Discord Search |
| GitHubプロジェクト名 | `discord-search` |
| 種類 | Discord Bot |

## 解決する課題

- Discord標準検索が使いづらい
- 欲しい会話がどこにあるかわからない
- キーワードが思い出せないと検索できない

## 提供する価値

- 自然言語で会話を検索できる（例:「先月の経理関連の話」）
- 曖昧な記憶から会話を見つけられる
- 検索結果からDiscordの該当メッセージにジャンプできる

## 主な機能

- 自然言語検索（セマンティック検索）
- 日付・期間での絞り込み
- 発言者での絞り込み
- 添付ファイルの有無での絞り込み
- 添付ファイル内容の検索
- Jumpリンク付き検索結果表示

## 技術スタック（予定）

| 項目 | 技術 |
|------|------|
| 検索エンジン | Gemini API File Search Tool |
| Bot | Discord.py / Discord.js |
| データ取得 | Discord API |
