# ペルソナ定義

## ターゲット

| 項目 | 内容 |
|------|------|
| 種別 | B2B（社内ツール） |
| 規模 | 2人の会社 |
| ユーザー | 自分と妻 |

## ペルソナ

### ユーザー1: 開発者（自分）

- **役職**: 開発者
- **立場**: プロジェクト全般を担当
- **課題**: 過去の会話を探すのに時間がかかる
- **ゴール**: 素早く必要な情報を見つけたい

### ユーザー2: 妻

- **役職**: 非エンジニア
- **立場**: プロジェクトメンバー
- **課題**: Discordの検索機能が使いこなせない
- **ゴール**: 簡単な操作で過去の会話を見つけたい

## 利用シーン

1. 「この前話したあの件、どこだっけ？」
2. 「先月の経理の話を確認したい」
3. 「添付したPDFの内容から検索したい」
4. 「○○について話したスレッドに戻りたい」
